# HarmonyHttpClient

#### Description
鸿蒙上使用的Http网络框架，里面包含纯Java实现的HttpNet，类似okhttp使用，支持同步和异步两种请求方式；还有鸿蒙版retrofit，和Android版Retrofit相似的使用，解放双手搬优雅使用注解、自动解析json
